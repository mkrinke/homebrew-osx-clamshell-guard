# homebrew-osx-clamshell-guard

Homebrew tap for [osx-clamshell-guard](https://github.com/mkrinke/osx-clamshell-guard).

## Install

```bash
brew tap mkrinke/osx-clamshell-guard
brew install osx-clamshell-guard
sudo brew services start osx-clamshell-guard
```

See the [main repository](https://github.com/mkrinke/osx-clamshell-guard) for documentation.
